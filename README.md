# SmartAgro_Advisor

Smart Agro Advisor is a unified, web-based platform designed to bridge this technological gap. The system combines Machine Learning (ML), Deep Learning (CNN), and real-time weather data to deliver three core services to farmers:

•	Crop Recommendation: A ML-based module that suggests the most suitable crop based on soil nutrient levels (Nitrogen, Phosphorous, Potassium), pH, rainfall, temperature, and humidity obtained from soil laboratory tests.

•	Plant Disease Detection: A Convolutional Neural Network (CNN) model that analyses uploaded leaf images and identifies diseases, offering instant treatment guidance.

•	Weather Forecast: Integration with the OpenWeatherMap API to provide real-time and 7-day weather forecasts, helping farmers plan irrigation, sowing, and harvesting activities.

The frontend is built using HTML, CSS, and JavaScript, offering a clean and farmer-friendly interface. The backend intelligence is powered by Python and Flask, serving the ML and CNN models through REST APIs. MongoDB is used as the database to store user data, prediction history, and feedback.

By combining modern AI/ML techniques with a practical web interface, Smart Agro Advisor empowers farmers to make data-driven decisions independently, reducing dependency on intermediaries and improving agricultural outcomes.
